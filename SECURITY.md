# Security

If you believe you have identified a security vulnerability in this project, please report it responsibly. **Do not open a public issue.**

## Reporting a Vulnerability

Send your report to [support@astravue.com](mailto:support@astravue.com) with the following details:

* A description of the vulnerability and its potential impact
* Steps to reproduce the issue
* Any relevant logs, screenshots, or proof-of-concept code

Our team will acknowledge your report within **48 hours** and provide an estimated timeline for a fix.

## Scope

This policy applies to the Astravue MCP Server endpoint (`api.astravue.com/mcp`) and the contents of this repository.

## Disclosure Policy

We follow coordinated disclosure. We ask that you give us reasonable time to investigate and address the issue before making any information public. We will credit reporters in the release notes unless anonymity is requested.
